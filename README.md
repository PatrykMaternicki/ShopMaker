# ShopMaker
The webshop with use turingAPI and Stripe API.

# Use:
- Vuex
- Vue-router
- Vuetify
- TuringAPI
- StripeAPI

# Features

- Search product
- Search for category
- Search for departaments
- Login
- Register
- Cache(LocalStorage)
- Make order
- Payment System
